Font Rasterizer

これは何？

WASM + WebGPU で書かれた Font Rasterizer のデモです。

操作方法

F11(or ダブルクリック, ダブルタップ) : フルスクリーン切り替え
シングルクリック(or シングルタップ) : 拡大
スワイプ(上下) : スクロール
スワイプ(左右) : 回転
